# axios classification example

`axios-classification`를 사용한 예시입니다. :)<br/>

## 🤹‍♂️ How to use

```bash
# clone repository
$ git clone https://github.com/gingaminga/axios-classification.git

# 폴더 이동
$ cd example

# 시작
$ npm run dev
```
